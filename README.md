# Explainable AI for Detecting Bias and Improving Fairness in Credit Scoring Systems

## Project Overview

This repository contains the implementation developed as part of the MSc Data Science dissertation titled **"Explainable AI for Detecting Bias and Improving Fairness in Credit Scoring Systems."**

The project proposes a machine learning framework that predicts credit risk while improving transparency and fairness in automated decision-making. Explainable AI techniques such as **SHAP** and **LIME** are used to interpret model predictions, and fairness metrics are applied to identify potential bias in credit scoring models.

---

## Objectives

* Develop machine learning models for credit risk prediction.
* Improve model transparency using Explainable AI (XAI).
* Detect and evaluate bias using fairness metrics.
* Compare different classification models based on prediction performance and fairness.
